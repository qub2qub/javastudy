Приведем формальные правила оценки истинности предикатов, использующих параметры ANY|SOME и ALL.

1)    Если определен параметр ALL или SOME/ANY и все результаты сравнения значения выражения и каждого значения, полученного из подзапроса, являются TRUE, истинностное значение равно TRUE.
2)    Если результат выполнения подзапроса не содержит строк и определен параметр ALL, результат равен TRUE. Если же определен параметр SOME/ANY, результат равен FALSE.
3)    Если определен параметр ALL и результат сравнения значения выражения хотя бы с одним значением, полученным из подзапроса, является FALSE, истинностное значение равно FALSE.
4)    Если определен параметр SOME/ANY и хотя бы один результат сравнения значения выражения и значения, полученного из подзапроса, является TRUE, истинностное значение равно TRUE.
5)    Если определен параметр SOME/ANY и каждое сравнение значения выражения и значений, полученных из подзапроса, равно FALSE, истинностное значение тоже равно FALSE.
6)    В любом другом случае результат будет равен UNKNOWN.
